Groovy Movies
Danielle Gray & Carly Kraft

A database of movie information for those who are a little unsure of what they want to watch.

Web Serivces / API used
  • The Movie Database (TMDb)
  • D3 - Data Driven Documents
Full list of Possible features
  • Search for movies
  • Results have gallery of movie covers with brief information on hover
  • Another page gives you full detail
  • Graphs using data such as budget versus revenue
  • Charts using data such as release locations
  • Watch movie trailers
  • See still shots from the movie
  • See movies by section such as 'now playing', 'top rated', 'popular now', etc.
  • See show times for the 'now playing' section
  • Buy your tickets for a movie based on your location
